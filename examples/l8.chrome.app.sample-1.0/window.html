Send commands to the L8 writing hex strings.

For example:
- "4B-00-00-0F" To set SuperLED in red color
- "48" To query the tempearture
- "81-02-01-30-00" To run the Multicolor Light App
- "82" To stop an App


* Write just the command Payload, the SDK function will add the header (AA55), the length and the CRC


Visit www.l8smartlight.com/dev/slcp/1.0/ for commands documentation
